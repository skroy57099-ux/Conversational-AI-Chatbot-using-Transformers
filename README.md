# Conversational-AI-Chatbot-using-Transformers
This project is an interactive AI-powered chatbot built using Microsoft DialoGPT-medium, capable of generating human-like responses based on conversation history. The chatbot demonstrates practical applications of Natural Language Processing (NLP) and conversational AI using the transformers library.

The goal of this project is to create a chatbot that:
- Understands user input
- Maintains conversation context
- Generates personalized and coherent replies
- Displays responses in a visually enhanced UI cell (Jupyter Notebook)

It uses Hugging Face Transformers, PyTorch, and DialoGPT to simulate real-time AI conversations.
